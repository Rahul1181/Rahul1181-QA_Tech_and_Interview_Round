# Interview Ready Automation Questions

## Core Java Coding Questions
1. Write a Java program to reverse a given string without using the StringBuffer class.
2. Write a program to find if a given integer or string is a palindrome.
3. Write a Java program to swap two numbers without using a third variable.
4. Write code to find duplicate or non-duplicate elements in an array or an ArrayList.
5. Write a program to find a prime number.
6. Explain the output of a Java program containing static blocks, constructors, and methods, demonstrating your understanding of the execution order.
7. Implement a singleton class in Java.
8. Write sample code to demonstrate the use of HashMap and Hashtable

## Selenium and Automation Coding/Scenario Questions
1. These questions focus on the practical application of Selenium WebDriver with Java. 
2. Write Selenium Java code to handle dynamic dropdowns or auto-complete lists.
3. Provide a code snippet to take a screenshot in Selenium WebDriver.
4. How do you handle synchronization issues? Provide code for an explicit wait that waits for an element to be clickable or visible.
5. Write code to scroll to a specific element on a page that loads asynchronously using JavaScript Executor.
6. Write the XPath expression to locate a specific cell in a web table based on the value of a neighboring cell.
7. How do you enter text without using sendKeys()? (Answer: using JavaScript Executor).
8. Write the code for reading data from an Excel file using the Apache POI library.
9. Write a sample WebDriver script for a basic login functionality (e.g., for Gmail).
10. How do you handle multiple windows or tabs? Write the code using getWindowHandle() and getWindowHandles() methods.

## Framework and Methodology Questions
1. You will also need to explain how you apply coding principles within a testing framework.
2. Explain how you have used Object-Oriented Programming (OOP) concepts (inheritance, encapsulation, polymorphism, abstraction) in your automation framework.
3. Explain the Page Object Model (POM) structure and why you use an object repository.
4. How do you achieve parallel execution of tests using TestNG or Maven?
5. How do you integrate your test automation into a CI/CD pipeline (e.g., Jenkins)?
6. Describe your approach to handling flaky tests and the debugging steps you take when a test fails in the CI environment.
